Amazon Dynamo-style Replicated Key-Value Storage

Amazon Dynamo-style Replicated Key-Value Storage

Java Android

  • Implemented a simplified version of Amazon DynamoDB which stores key-value pairs in its partitioned nodes
  • Most recent data-writes were returned (and not stale data copies) even under the failure of nodes in the system
  • Data consistency was provided even in the presence of concurrent reads and writes and failure of nodes.
  • The node failure tolerance was a achieved by replicating data in other alive nodes which were next to failed node in the chord ring.

Java Android

Large scale Text processing with Hadoop Map Reduce and Apache Spark

Large scale Text processing with Hadoop Map Reduce and Apache Spark

Java Python

  • The goal was to find the word count and perform Inverted indexing of each word in the classical Latin text files provided by the researchers of Classics Department at UB.
